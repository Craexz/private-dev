Who am I?

I'm Ben. I'm a 14 year old software developer. I know 7 different coding languages, all of them self-taught!


Open Source Projects

Calendar App

One of my projects is an app I developed in pure python, html, and css. It's a site where users can sign up and login, then add and remove assignments to keep track of them.

D.O.T.S.

I developed a fully in-browser nodejs and javascript game. Users can see others move in real time. There is also a chat.

The Shrine

In pure python, html, and css, plus a week of work, I created the shrine. A platform for users to create their own accounts, and sign in seamlessly. I created this system from scratch. The users have the option to post products that can be bought by other users with a fake currency. Users can also make payments to others.

Discord Kahoot Flooder

I developed a program to use multiple "workers" that constantly send bots to a kahoot game. I integrated this with a discord bot, so all you have to do to use it, is create a discord bot, then run it and use a command to flood a game!

Stress Tester

In my time developing web servers for my projects, I've realized that there are lots of vulnerabilities. I created this tool to test my own projects. It continuously sends packets to the server to see if it can crash it. This is for development and pen-testing only.

IP Puller

I created this tool to showcase how easily you can obtain someones information. This tool runs a script from an api that gets your ip. It then sends it to the server. This tool is merely a test of how insecure devices are.

Private Projects

Youtube Scraper with Automatic Facial Recognition and Processing

This program isn't open source because of the work that went into it. The program has a simple task: get videos of people's faces, and use them to train a neural network. The program succesfully scrapes videos from youtube that are tagged to have people's faces in them. It then adds them to a queue and processes them one by one, then outputs the data to be used to train a neural network.

What I do

I've been a self-taught developer for a while, but plan to collaborate with others in the future.

Javascript

One day I decided that it would be cool to create a little game within a bookmark. I started writing little scripts where it drew a little dot which you moved with your arrow keys. This expanded into what I know today.

HTML

In sixth grade, my friend group had an idea. We'd create a website with funny information about our friends. So, I taught myself html and used it to create a website, which was surprisingly responsive and good looking! This has expanded my knowledge and brought it to what it is today.

Python

It started with a "Hello World" program. I expanded my knowledge of python over time, and eventually got to the level I am today. I started out with creating a program to solve simple problems like solving the area of a shape, and now, my projects are much more advanced.

NodeJS

Lua

CSS
